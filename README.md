# 📖 Leaderboard <a name="about-project"></a>

**Leaderboard** In this project, I build a simple HTML structure of a Leaderboard app that you can record names and respective scores.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

<details>
  <summary>Client</summary>
  <ul>
    <li>HTML</li>
    <li>JavaScript</li>
    <li>CSS</li>
    </ul>
</details>

<p align="right">(<a href="#readme-top">back to top</a>)</p>

### Key Features <a name="key-features"></a>

- Leaderboard Section - This part displays the names and scores respectively in order of highest to lowest.

- Add Name Section - Adds a persons name to the leaderboard list.

- Add Scores Section - Adds a person's score to the leaderboard list.
